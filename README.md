# 🍜 Foodstagram

Foodstagram adalah aplikasi berbagi kuliner Nusantara berbasis PHP dan MySQL.

## Fitur

* Login & Register
* Upload Foto Makanan
* Feed Kuliner
* Search Makanan
* Lokasi Kuliner
* Google Maps
* Responsive Design

## Teknologi

* PHP Native
* MySQL
* HTML5
* CSS3
* XAMPP

## Cara Menjalankan

1. Clone repository

```bash
git clone https://github.com/username/foodstagram.git
```

2. Pindahkan ke folder htdocs

3. Jalankan Apache dan MySQL

4. Import file database.sql ke phpMyAdmin

5. Buka browser

```text
http://localhost/foodstagram
```
